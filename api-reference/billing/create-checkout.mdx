---
title: "Create Checkout Session"
openapi: "POST /api/stripe/create-checkout-session/"
description: "Create a Stripe hosted checkout session to subscribe to a plan."
---

Creates a Stripe Checkout session for a subscription plan and returns the redirect URL. The user should be forwarded to `checkout_url` to complete payment on Stripe's hosted page.

## Request Body

<ParamField body="plan_id" type="string" required>
  UUID of the subscription plan to purchase.
</ParamField>

<ParamField body="success_url" type="string">
  URL to redirect to after successful payment. Defaults to the app's billing success page.
</ParamField>

<ParamField body="cancel_url" type="string">
  URL to redirect to if the user cancels checkout. Defaults to the app's billing page.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` when the checkout session was created.
</ResponseField>

<ResponseField name="data.checkout_url" type="string">
  Stripe hosted checkout URL. Redirect the user to this URL to complete payment.
</ResponseField>

<RequestExample>
```json
{
  "plan_id": "plan-uuid-here",
  "success_url": "https://app.webyes.com/billing?status=success",
  "cancel_url": "https://app.webyes.com/billing"
}
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "checkout_url": "https://checkout.stripe.com/pay/cs_live_..."
  }
}
```
</ResponseExample>

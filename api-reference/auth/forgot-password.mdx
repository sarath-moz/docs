---
title: "Forgot Password"
openapi: "POST /api/auth/forgot-password"
description: "Request a password reset link sent to the user's email."
---

Sends a password-reset email to the provided address if an account exists. The response is identical regardless of whether the email is registered — this prevents email enumeration attacks.

<Note>
  The reset link expires after **24 hours**.
</Note>

## Request Body

<ParamField body="email" type="string" required>
  The email address associated with the account.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Always `true` (even if the email is not registered).
</ResponseField>

<ResponseField name="message" type="string">
  Generic confirmation message.
</ResponseField>

<RequestExample>
```json
{
  "email": "dev@webyes.com"
}
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "message": "If an account exists for this email, a password reset link has been sent."
}
```
</ResponseExample>

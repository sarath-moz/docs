---
title: "Reset Password"
openapi: "POST /api/auth/reset-password"
description: "Complete a password reset using the token from the reset email."
---

Sets a new password using the one-time token from the Forgot Password email.

<Warning>
  Reset tokens are **single-use** and expire after **24 hours**. Expired or already-used tokens return `400`.
</Warning>

## Request Body

<ParamField body="token" type="string" required>
  The password-reset token from the email link.
</ParamField>

<ParamField body="new_password" type="string" required>
  The new password. Must be at least **8 characters**.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` when the password has been updated.
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message.
</ResponseField>

<RequestExample>
```json
{
  "token": "eyJhbGciOiJIUzI1NiJ9.resettoken...",
  "new_password": "myNewSecurePass99"
}
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "message": "Password reset successfully. You can now log in with your new password."
}
```
</ResponseExample>

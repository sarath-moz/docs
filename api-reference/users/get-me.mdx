---
title: "Get Current User"
openapi: "GET /api/user/me/"
description: "Retrieve the authenticated user's profile, organisation, and permissions."
---

Returns the full profile of the currently authenticated user, including their organisation membership and permission slugs.

## Response

<ResponseField name="success" type="boolean">
  `true` on a successful request.
</ResponseField>

<ResponseField name="data.user" type="object">
  User profile — id, email, name, status, `is_from_suit` flag, `email_verified`.
</ResponseField>

<ResponseField name="data.organisation" type="object">
  The user's current organisation — id, name, status, page limits.
</ResponseField>

<ResponseField name="data.permissions" type="array">
  List of permission slugs granted to this user. Example slugs:
  `settings:website:add`, `scan:full_scan:add`, `settings:team:delete`.
</ResponseField>

<RequestExample>
```bash
curl https://api.webyes.com/api/user/me/ \
  -H "Authorization: Bearer <token>"
```
</RequestExample>

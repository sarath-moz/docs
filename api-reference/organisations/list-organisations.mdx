---
title: "List Organisations"
openapi: "GET /api/organisation/"
description: "List all organisations the authenticated user belongs to."
---
**`GET`**
```
/api/organisation/
```


Returns all organisations the currently authenticated user is a member of, along with their role in each.

## Response

<ResponseField name="success" type="boolean">
  `true` on a successful request.
</ResponseField>

<ResponseField name="data" type="array">
  Array of organisation objects. Each includes `id`, `organisation_name`, `status`, `role`, and `created_time`.
</ResponseField>

<RequestExample>
```bash
curl https://api.webyes.com/api/organisation/ \
  -H "Authorization: Bearer <token>"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": [
    {
      "id": "org-uuid-1",
      "organisation_name": "Acme Web Agency",
```

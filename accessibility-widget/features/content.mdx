## Content Features

Content features help users adjust text size, spacing, fonts, and other content-related settings for better readability.

## Text Size Features

### Bigger Text

Increase font size with three adjustable levels.

**Levels:** 0 (off), 1 (small increase), 2 (medium increase), 3 (large increase)

**Enable:**
```javascript
modules: {
  content: {
    biggerText: { enabled: true }
  }
}
```

### Text Magnifier

Magnify text when hovering over it. Useful for reading small text without changing the entire page layout.

**Enable:**
```javascript
modules: {
  content: {
    textMagnifier: { enabled: true }
  }
}
```

```javascript
window._clvA11yConfig = {
  modules: {
    content: {
      biggerText: { enabled: true },
      cursor: { enabled: true },
      dyslexicFont: { enabled: true },
      letterSpacing: { enabled: true },
      lineHeight: { enabled: true },
      readableFont: { enabled: true },

      textMagnifier: { enabled: true },

  }

```

## Feature Interactions

Some content features work well together:

- **Bigger Text + Line Height**: Increases both size and spacing for maximum readability
- **Dyslexic Font + Letter Spacing**: Combines dyslexia-friendly font with increased spacing
- **Readable Font + Bigger Text**: Uses readable font at larger size
- **Highlight Links + Bigger Text**: Makes links both larger and more visible

## Best Practices

1. **Enable Core Features**: At minimum, enable `biggerText` and `readableFont` for basic accessibility
2. **Font Choice**: Consider your website's design when enabling font features
3. **Level Testing**: Test different levels to ensure they work well with your content
4. **Performance**: Font features may require loading additional font files

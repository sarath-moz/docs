- Stop Animations

## Feature Levels

Some features support multiple levels of adjustment:

- **Level 0**: Off/Default
- **Level 1**: Light adjustment
- **Level 2**: Moderate adjustment
- **Level 3**: Maximum adjustment

Features with levels:
- Bigger Text (3 levels)
- Cursor (3 levels)
- Line Height (3 levels)
- Letter Spacing (3 levels)
- Align Text (3 levels)
- Invert Colors (3 levels)
- Brightness (3 levels)
- Contrast (3 levels)
- Grayscale (3 levels)
- Saturation (3 levels)

## Enabling Features

Configure which features are available:

```javascript
window._clvA11yConfig = {
  modules: {
    color: {
      brightness: { enabled: true },
      invertColors: { enabled: true },
      contrast: { enabled: true },
      grayScale: { enabled: true },
      saturation: { enabled: true }
    },
    content: {
      biggerText: { enabled: true },
      cursor: { enabled: true },
      dyslexicFont: { enabled: true },
      letterSpacing: { enabled: true },
      lineHeight: { enabled: true },
      readableFont: { enabled: true },
      alignText: { enabled: true },
      textMagnifier: { enabled: true },
      highlightLinks: { enabled: true }
    },
    orientation: {
      readingLine: { enabled: true },
      keyboardNavigation: { enabled: true },
      highlightTitles: { enabled: true },
      readingMask: { enabled: true },
      hideImages: { enabled: true },
      highlightAll: { enabled: true },
      readPage: { enabled: true },
      muteSounds: { enabled: true },
      stopAnimations: { enabled: true }
    }
  }
};
```

## Feature Persistence

User preferences are automatically saved to localStorage and persist across page loads. Users can reset all settings using the reset button in the widget.

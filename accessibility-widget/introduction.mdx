## What is the Accessibility Widget?

The WebYes Accessibility Widget is a powerful, customizable accessibility solution that helps make your website more accessible to users with disabilities. It provides a wide range of features including visual adjustments, reading aids, keyboard navigation, and more.

## Key Features

### Accessibility Profiles

Pre-configured profiles that activate multiple features at once:

- **Vision Impaired Mode** - Enhances text readability and visual clarity
- **Seizure Safe Profile** - Reduces flashing and motion
- **ADHD Friendly Mode** - Focused browsing with reduced distractions
- **Blindness Mode** - Enhanced screen reader support
- **Epilepsy Safe Mode** - Prevents seizure triggers

### Content Adjustments

Fine-tune text and content presentation:

- **Bigger Text** - Increase font size with 3 levels
- **Readable Font** - Switch to a more readable font family
- **Dyslexic Font** - Use OpenDyslexic font for better readability
- **Line Height** - Adjust spacing between lines
- **Letter Spacing** - Control character spacing
- **Text Alignment** - Align text left, center, or right
- **Text Magnifier** - Magnify text on hover
- **Highlight Links** - Make links more visible

### Color Adjustments

Modify colors for better visibility:

- **Invert Colors** - Invert the color scheme
- **Brightness** - Adjust screen brightness
- **Contrast** - Increase or decrease contrast
- **Grayscale** - Convert to grayscale
- **Saturation** - Adjust color saturation

### Orientation & Navigation

Improve navigation and reading experience:

- **Reading Line** - Highlight the current reading line
- **Reading Mask** - Focus on specific content areas
- **Keyboard Navigation** - Enhanced keyboard support
- **Highlight Titles** - Make headings more visible
- **Highlight All** - Highlight all important elements
- **Read Page** - Text-to-speech functionality
- **Hide Images** - Hide images for faster loading
- **Mute Sounds** - Disable audio
- **Stop Animations** - Pause all animations

## Browser Support

The widget works on all modern browsers:

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Opera (latest)
